

 **Friends KeenKeeper App**
      **Description**
   -Friend Tracker is a modern web application that helps you manage and maintain your personal relationships. You can track your interactions with friends, set communication goals, and visualize your connection habits through analytics.


  **Technologies**
   Next.js (App Router)
   React.js
   Tailwind CSS
   Recharts
   React Icons
   LocalStorage (for data persistence)


  **Key Features**
  1.Friend Management
    View all friends in a clean card layout
    Detailed profile page for each friend
    Track days since last contact

   2.Interaction Tracking
    Log interactions (Call, Text, Video)
    Timeline page to view all history
    Filter interactions by type

   3.Analytics Dashboard
    Dynamic Pie Chart showing interaction types
    Real-time updates based on user activity
    Visual representation of communication patterns


  **Additional Features**
    Responsive design (Mobile, Tablet, Desktop)
    Toast notifications for actions
    Custom 404 page
    Loading states for better UX
